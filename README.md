# 📸 Photography Studio & Gallery Showcase

A modern, visually stunning, and **100% responsive web design project** crafted specifically for a professional photography studio. This project focuses on a dynamic layout and smooth user experience using pure CSS.

## ✨ Features
- **Dynamic Layout:** Built using modern CSS layout techniques (Grid & Flexbox) for fluid resizing.
- **Pure CSS:** No heavy frameworks used—pure, lightweight, and optimized CSS3.
- **Responsive Design:** Completely optimized for Desktops, Tablets, and Mobile devices.
- **Interactive Gallery:** High-quality image showcase with smooth hover effects.

## 🛠️ Tech Stack
- **HTML5:** Semantic structure for better SEO and accessibility.
- **CSS3:** Advanced styling, custom properties (variables), and smooth transitions.
